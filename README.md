# Autotech Power BI Theme

## Instrucciones de Instalación

1.  **Descarga el archivo**: Asegúrate de tener el archivo `Autotech Power BI Theme.json` en tu computadora.
2.  **Abre Power BI**: Inicia Power BI Desktop y abre tu reporte.
3.  **Ve a la pestaña "Ver" (View)**: En la cinta de opciones superior.
4.  **Expande Temas**: Haz clic en la flecha hacia abajo en la galería de temas.
5.  **Buscar Temas**: Selecciona la opción **"Buscar temas" (Browse for themes)**.
6.  **Selecciona el Archivo**: Busca y selecciona `Autotech Power BI Theme.json`.
7.  **¡Listo!**: El tema se aplicará automáticamente con el fondo, colores y estilos profesionales.
